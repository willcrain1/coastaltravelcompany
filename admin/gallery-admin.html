One-time setup required. Three things need to be in place before you create galleries:
  1. Deploy the Cloudflare Worker — log in to dash.cloudflare.com → Workers & Pages → Create Worker → paste the contents of cloudflare-worker.js → Deploy. Copy the *.workers.dev URL and paste it into the Worker URL setting below.
  2. GitHub Pages — gallery.html and client-gallery.html are already live on coastaltravelcompany.com from the GitHub repo. No NAS hosting needed.
  3. Fill in the three URL settings below, then create galleries using a Synology Photos share link.
